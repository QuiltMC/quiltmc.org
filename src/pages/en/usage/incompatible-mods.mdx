This page lists mods which are incompatible with Quilt. You can hover the symbol on the left to see the current status of the bug report tracking the incompatibility, and click the link icon to visit the report. If you find an incompatible mod that is not on the list, please tell us [on Discord](https://discord.quiltmc.org). You can also submit a Pull Request to update the data [on GitHub](https://github.com/QuiltMC/quiltmc.org?tab=readme-ov-file#adding-an-incompatible-mod).
