import IncompatibleModList from "@parts/IncompatibleModList.astro"
import { getCollection } from "astro:content";

export const rawData = await getCollection("incomatibleMods")
export const data = rawData.sort((a, b) => a.data.name.localeCompare(b.data.name))


This page lists mods which are known to be incompatible with Quilt. You can hover the symbol on the left to see the current status of the issue, and click the link icon to go to the tracking issue. If you believe this list is missing anything, please feel free to tell us [on Discord](https://discord.quiltmc.org) You can also directly [update the data on GitHub](https://github.com/QuiltMC/quiltmc.org/blob/main/public/api/v1/incompatible-mods.json).

## Requires a Workaround
This is the simplest form of problems, the issue can simply be fixed by following the instructions provided.

<IncompatibleModList mods={data} type="WORKAROUND" />

## Non-Fatal Errors or Broken Features
These mods are not compatible with Quilt, but they do not break the game or other mods. They may have broken features or other errors that do not crash the game.
